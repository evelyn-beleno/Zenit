# Zenit - Sitio web estático de Activewear

Sitio web estático para la marca Zenit, con enfoque en ropa deportiva y activewear para mujeres.

## Estructura del proyecto

- `index.html` — Página de inicio con banner, novedades y sección de marca.
- `productos.html` — Página de catálogo de productos con tarjetas de prendas y video de guía.
- `contactenos.html` — Página de contacto con formulario y detalles de ubicación.
- `css/` — Carpeta de estilos CSS.
  - `index.css`
  - `productos.css`
  - `contactenos.css`
- `img/` — Carpeta para imágenes del sitio.

## Tecnologías

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- Google Fonts (`Montserrat`)

## Cómo usar

1. Abre el archivo `index.html` en tu navegador.
2. Navega entre `index.html`, `productos.html` y `contactenos.html`.

## Recomendaciones

- Mantén las imágenes de `img/` optimizadas para mejorar tiempos de carga.
- Si vas a editar estilos, usa `css/index.css` para elementos generales y `css/productos.css` / `css/contactenos.css` para estilos específicos de página.
